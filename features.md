# Simple Location Manager - Features

## 📍 Core Location Tracking

- **Save Anywhere**: Instantly save your current location with a custom name and description.
- **Auto-Discovery**: Automatically detects and records the **District** and **Sub-District** for every saved spot.
- **Smart Defaults**: Auto-generates descriptions with timestamps if left blank (configurable).
- **Duplicate Prevention**: Warns you if you try to save a location too close to an existing one (Distance threshold customizable).

## 📂 Organization & UI

- **clean Interface**: A native-feeling, resizeable, and moveable UI window.
- **Smart Grouping**: Locations are automatically grouped by District -> Sub-District.
- **Collapsible Groups**: Expand or Collapse separate districts or control the default global state.
- **Favorites**: Star your most-used spots for quick access in a dedicated top section.
- **Search**: Real-time filtering by name, description, district, or sub-district.

## 💾 Import / Export

- **Clipboard Sharing**: Export any location, district, or your entire database to a compact text string.
- **Easy Import**: Import locations from friends or backup strings with a simple Paste + Click.
- **Conflict Handling**: Smart detection of duplicates during import.

## 🛠️ Tools & Customization

- **Navigation**: Set a custom game Map Pin to any saved location with one click.
- **Teleport (Lazy Mode)**: Optional cheat feature to instantly teleport to saved locations (Disabled by default).
- **Customizable UI**:
  - Toggle display of Coordinates and District details in the list.
  - Adjustable Duplicate Warning Distance (0m - 50m).
  - Configurable Default Group State (Start Expanded or Collapsed).
- **Safety**: "Are you sure?" confirmation modals for destructive actions.
