AMM IMPORT FOLDER
=================

Due to Cyber Engine Tweaks (CET) sandbox restrictions, Simple Location Manager cannot directly access the proper Appearance Menu Mod folder.

To import locations from AMM:
1. Navigate to: bin/x64/plugins/cyber_engine_tweaks/mods/AppearanceMenuMod/User/Locations
2. Copy the .json files you want to import.
3. Paste them into this folder (import/).
4. In the game, open Simple Location Manager > Locations Tab > AMM Bulk Import.
5. Click "Scan Directory & Import".
